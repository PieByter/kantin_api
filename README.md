# Nama Project

## Instalasi

```bash
go mod tidy
go run main.go
```

## API Endpoints

| Method | Endpoint  | Deskripsi       |
| ------ | --------- | --------------- |
| POST   | /register | Register user   |
| POST   | /login    | Login user      |
| GET    | /users    | List semua user |
| ...    | ...       | ...             |

## Struktur Folder

- `/controllers` : Logic API
- `/models` : Struktur data
- `/routes` : Routing API
- `/config` : Koneksi DB
